## ✨ Sobre o Projeto

Site institucional da **Jobelly Bordados**, desenvolvido com React + TypeScript + Tailwind CSS. Apresenta os serviços, brindes promocionais, portfólio e canais de contato da empresa.

## 🎁 Seções

| Seção | Descrição |
|-------|-----------|
| **Início** | Carrossel hero com destaques da empresa |
| **Sobre** | História e diferenciais da Jobelly |
| **Serviços** | Bordados, chaveiros, brindes e laser |
| **Brindes** | Galeria completa com filtros por categoria |
| **Orçamento** | Formulário rápido para pedidos |
| **Portfólio** | Trabalhos realizados |
| **Contato** | Endereço, telefone e mapa |

## 🛠️ Tecnologias

- **React 18** + **TypeScript**
- **Vite** (build tool)
- **Tailwind CSS** (estilização)
- **shadcn/ui** (componentes)
- **Framer Motion** (animações)

## 🚀 Rodar localmente

```bash
# 1. Clone o repositório
git clone https://github.com/ramerico/JobellyBordados.git

# 2. Acesse a pasta
cd JobellyBordados

# 3. Instale as dependências
npm install

# 4. Inicie o servidor de desenvolvimento
npm run dev
```

Acesse em: `http://localhost:5173`

## 📦 Build para produção

```bash
npm run build
```

Os arquivos serão gerados na pasta `dist/`.

## 🌐 Deploy

O site é hospedado na **Vercel** com deploy automático a cada push na branch `main`.

[![Deploy with Vercel](https://vercel.com/button)](https://vercel.com/new/clone?repository-url=https://github.com/ramerico/JobellyBordados)
